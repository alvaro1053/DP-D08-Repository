1. Hemos considerado que las questions no se pueden ni editar ni añadir nuevas una vez creado el rendezvous. Si se pudieran editar las
replies no tendrían sentido.Si se pudieran añadir nuevas, habría usuarios que se quedan sin contestarlas puesto que los usuarios
están obligados a responder a las preguntas creadas cuando van a asistir a un Rendezvous.
2. Hay un bug en la vista de crear Rendezvous. Añadir y quitar questions dinámicamente está hecho con jequery pero cuando decides no poner questions y eliminas todos los inputs, no puedes añadirlos a no ser que se recargue la página.
3. Cuando eres menor de edad no se te muestran los rendezvous calificados para mayor de edad aunque pongas el filtro de "mostrar todo".
4.En Acme-Rendezvous no aparece nada sobre contestar a las respuestas de un comentario, por lo que hemos optado porque solo se puedan contestar a los comentarios.
5.Hemos optado por poner los términos y condiciones con javascript.
6.Hemos considerado que, aunque el user guarde una dirección postal, no es apropiado mostrarla públicamente cuando se muestra la información de un usuario. Por lo tanto, hemos optado por mostrar la dirección sólo si un usuario está viendo
su propia información.
